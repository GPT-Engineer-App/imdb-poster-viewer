## Tech stack

This project is built with React with Daisy UI and Tailwind CSS.

- Vite
- React
- Daisy UI

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/imdb-poster-viewer.git
cd imdb-poster-viewer
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
